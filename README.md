# Lapis Api

Runtime API for Lapis KSP.

[![Maven Central](https://img.shields.io/maven-central/v/io.github.recrafter/lapis-api.svg?label=Maven+Central&style=for-the-badge)](https://central.sonatype.com/artifact/io.github.recrafter/lapis-api) [![License: MIT](https://img.shields.io/static/v1?label=License&style=for-the-badge&message=MIT&color=yellow)](https://spdx.org/licenses/MIT)

---

## Usage

This is a small Kotlin library required alongside [Lapis KSP](https://github.com/Recrafter/lapis) for code generation and runtime processing. For detailed installation and usage instructions, refer to the Lapis KSP documentation.
